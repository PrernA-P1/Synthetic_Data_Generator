# Synthetic Data Generator

This project is a web-based application developed using Python that generates synthetic datasets for multiple domains such as hospital, college, office, airport, and software agencies. The system allows users to create realistic sample data for testing, analysis, and educational purposes.

## Features
- Generate synthetic data for multiple domains
- Export data in CSV, Excel, and PDF formats
- User login and profile management
- Database integration for storing generated data
- Simple and interactive web interface

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database

## Purpose
The project helps users create sample datasets for practice, testing, and learning without using real-world sensitive data.
